# hw_01

Проект с ДЗ_1 по системам хранения и обработки данных. 

Решение в [notebook](./notebook.ipynb).

## Структура
```
.
├── assets - различные картинки и скриншоты.
├── dbml - схема баз данных в `.dbml`.
├── docker-compose.yaml - для локального запуска postgres.
├── notebook.ipynb - файл с решением
├── raw_data - исходные таблицы с данными.
├── README.md
├── requirements.txt
└── sql - содержит `.sql` скрипты для создания БД.
```
